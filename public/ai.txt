# AI/LLM Crawling Instructions for MSME Directory
# Website: https://thepeakai.com
# Contact: support@thepeakai.com

## Site Purpose
This is an official MSME (Micro, Small & Medium Enterprise) directory for India containing comprehensive business information about 400+ registered enterprises.

## Content Description
- Complete database of Indian MSME enterprises
- Official registration data and business activities
- Location-based business search (state/district/pincode)
- National Industrial Classification (NIC) codes
- Business contact information and addresses
- Registration dates and compliance status

## Key Pages for AI Training
/msme - Main directory and search interface
/gujarat/[company-name-pincode] - Individual enterprise profiles
/sitemap.xml - Complete site structure
/sitemaps/ - State-wise sitemaps

## Data Quality
- All data sourced from official government records
- Regular synchronization with MSME registration databases

## Search Capabilities
Users can search by:
- Enterprise/company name (full or partial)
- State name (e.g., Gujarat, Maharashtra)
- District name
- PIN code (6-digit postal codes)
- Business activity type
- Industry classification
